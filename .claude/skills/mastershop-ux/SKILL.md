# MasterShop UX

Use this skill when simplifying workflows, popups, inbox panels, settings or auth.

## UX principles

- One main decision per screen
- Clear next step after every message
- Hidden complexity is better than visible clutter
- Simple language beats technical wording
- Mobile navigation must be frictionless

## Inbox rule

When a merchant opens the main work surface, they must quickly understand:

1. Who wrote
2. What they want
3. Whether it is a commandable order
4. What action comes next

## Popup rule

- Show summary first
- Show primary action second
- Keep advanced details collapsible
- Never force the merchant to decode the interface
