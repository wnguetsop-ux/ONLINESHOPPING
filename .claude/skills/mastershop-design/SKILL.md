# MasterShop Design

Use this skill when redesigning the landing, inbox, order views, onboarding or shared UI.

## Visual direction

- Blend WhatsApp clarity, Stripe polish and SaaS practicality
- Warm premium surfaces, not cold minimalist emptiness
- Strong hierarchy, rich cards, soft motion
- Mobile-first, finger-friendly controls

## Required design behaviors

- Reuse strong existing assets before generating new ones
- Restore meaningful hover states and micro-interactions
- Keep images doing explanatory work, not decorative filler
- Keep marketing richer than back-office surfaces

## Shared visual rules

- Rounded corners should feel consistent
- Shadows should be soft but present
- Hover must give confidence
- Important actions must stand out immediately
