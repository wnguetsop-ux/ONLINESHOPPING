# Prompts Nano Banana / Gemini

## Objectif
Les visuels doivent expliquer le produit avant de decorer la page.

## Convention de fichiers
- `public/generated/landing-product-inbox-01.png`
- `public/generated/landing-product-order-01.png`
- `public/generated/landing-context-seller-cm-01.png`

## Prompt 1 - Inbox produit
```
Create a realistic SaaS product mockup for a WhatsApp order management app used by African merchants in Cameroon.
Show a clean dashboard with 3 columns:
left column inbox with customer conversations and unread badges,
center column full WhatsApp-like conversation thread,
right column customer card and draft order summary.
The product must feel credible, mobile-first, modern, practical, and readable.
Use French UI labels such as "Nouveau message", "Commande brouillon", "Client", "Reste a payer".
No futuristic sci-fi design, no abstract art, no fake impossible UI.
Bright daylight, warm trustworthy palette, premium SaaS presentation, realistic shadows.
```

## Prompt 2 - Message vers commande
```
Generate a realistic explanatory product visual for a landing page.
Show on the left a WhatsApp message from a customer asking for 2 products and delivery.
Show on the right the same request transformed into a structured draft order inside a SaaS interface.
The transformation must be obvious at first glance.
Target audience: non-technical merchants in Cameroon selling through WhatsApp.
French labels only. Practical and believable product scene.
```

## Prompt 3 - Contexte camerounais
```
Create a realistic lifestyle scene of a Cameroonian merchant using a smartphone and a laptop to manage WhatsApp customer orders.
The merchant is in a small but professional boutique environment with product shelves and delivery parcels.
The phone screen should suggest WhatsApp conversations and the laptop should suggest an order dashboard.
Keep the scene natural, local, useful, not staged, not generic corporate.
```

## Prompt 4 - Hero simple
```
Design a clean hero image for a landing page of a WhatsApp order management SaaS.
One merchant, one smartphone, one dashboard, one clear message:
messages become tracked orders.
Focus on clarity, trust, and business usefulness rather than visual effects.
```
